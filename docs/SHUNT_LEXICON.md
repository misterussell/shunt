# SHUNT LEXICON

This document contains approved vocabulary for generating content.

Terms here are not necessarily canon entities.

They are building blocks for locations, NPCs, factions, events, items, and dialogue.

Agents should prefer terms from this document before inventing new vocabulary.

---

# World Language Model

Shunt uses two overlapping vocabularies:

## The Physical World

Infrastructure, labor, transit, salvage, maintenance.

Examples:

* Pipe Market
* Relay Block
* Cooling Yard
* Transfer Hall
* Dock Union

## The Latticework

Signals, ghosts, echoes, traces, hidden identities, fragmented data.

Examples:

* Ghostdeck
* Black Signal
* Dead Trace
* Echo Court
* Whisper Syndicate

The tension between these vocabularies creates the setting's identity.

# Things To Avoid

Avoid technology names that sound like advertisements.

Poor Examples:

* HyperNet
* QuantumSync
* UltraLink
* CyberSphere
* OmniGrid X9

Technology should feel adopted, repaired, renamed, and passed between users.

People rarely use official names if a better nickname exists.

# Recurring Terms — The Liftworks

The Authority-run ascent checkpoint where the Underbelly meets the Midgrid, built
around old inter-strata freight lifts.

* **The Liftworks** — the checkpoint district itself ([Infrastructure] + Works).
* **The Risers** — the freight-lift hall; the way up between strata.
* **The Cold Stair** — the disused, unwatched service stair; the unofficial route up.
* **Transit Permit** — Authority-issued (or forged) clearance to ride the lifts.
* **Scan Arch** — the checkpoint reader that clears who passes.
* **Permit Registry** — the Latticework-wired records system behind the checkpoint.
